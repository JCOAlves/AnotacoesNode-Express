# Express
O Express é um framework web para Node.js que facilita e acelera a criação de aplicativos e APIs back-end, oferecendo recursos como roteamento, tratamento de requisições HTTP e middleware. Ele funciona como uma camada de abstração sobre o Node.js, que é o ambiente de runtime que executa JavaScript no servidor.  

## Funcionalidades principais do Express:
- **Roteamento:** Permite definir rotas para diferentes URLs e verbos HTTP (GET, POST, etc.).
- **Middleware:** Oferece uma forma de adicionar funções que executam antes que as rotas sejam finalizadas, permitindo a manipulação de requisições, respostas e a adição de novas funcionalidades.
- **Gerenciamento de requisições e respostas:** Simplifica o trabalho com requisições e respostas HTTP.
- **Integração com motores de visualização (view engines):** Ajuda a gerar conteúdo dinâmico, como páginas web, inserindo dados em templates.
- **Configuração:** Facilita a configuração de parâmetros comuns, como a porta de conexão do servidor. 

## Relação com o Node.js
- **Node.js:** É o ambiente de runtime que permite a execução do código JavaScript fora do navegador (no servidor). 
- **Express:** É um framework construído sobre o Node.js. Ele não substitui o Node.js, mas o complementa, fornecendo uma estrutura mais organizada e ferramentas para criar aplicações web de forma mais eficiente.
